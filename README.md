# Firebase 3 + Ionic 2 login via email &amp; password 
## Steps to install
<ul>
<li>git clone https://github.com/karma-emprendedor/firebase-login</li>
<li>cd firebase-login</li>
<li>npm install</li>
<li>ionic state restore</li>
<li>ionic serve</li>
</ul>
